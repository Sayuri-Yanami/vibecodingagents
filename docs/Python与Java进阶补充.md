# Python 与 Java 进阶补充题

## Python 列表和元组有什么区别？

列表可变，适合动态增删元素；元组不可变，通常更适合表达固定记录，也可以在元素均可哈希时作为字典键。元组不可变不代表其内部引用的可变对象也不可修改。

## Python 的可变默认参数有什么问题？

默认参数在函数定义时求值一次，而不是每次调用都重新创建。使用列表或字典作为默认值会让多次调用共享同一对象。通常使用 `None` 作为默认值，并在函数内部创建新对象。

## Python 中 is 和 == 有什么区别？

`==` 比较值是否相等，可由对象的相等方法自定义；`is` 比较是否为同一个对象。判断 `None` 应使用 `is None`。不能依赖小整数或字符串驻留现象用 `is` 比较值。

## Python 上下文管理器有什么作用？

上下文管理器在进入和退出代码块时执行确定性的准备与清理，典型语法是 `with`。它常用于文件、锁、事务和临时资源。可以通过实现 `__enter__`、`__exit__` 或使用 `contextlib` 创建。

## Python 垃圾回收机制是什么？

CPython 主要使用引用计数，对象引用数归零时通常立即释放；另外使用分代循环垃圾回收处理互相引用的容器对象。析构时机和行为不应被当作跨 Python 实现都一致的业务保证。

## Python 多进程和多线程如何选择？

I/O 密集任务常使用多线程或异步 I/O；CPU 密集的纯 Python 代码受 CPython GIL 限制，通常使用多进程。多进程具有更高启动和通信成本，选择时还要考虑数据共享、任务粒度和部署环境。

## Java String 为什么是不可变的？

String 创建后内容不能改变，这有利于字符串常量池复用、哈希值缓存、线程安全和安全边界。拼接大量字符串时应使用 `StringBuilder`，并避免在循环中反复创建中间字符串。

## Java 中 equals 和 hashCode 有什么约定？

如果两个对象 `equals` 相等，它们必须具有相同的 `hashCode`；反过来哈希值相同不代表对象相等。重写 `equals` 时通常必须同时重写 `hashCode`，否则在 HashMap 和 HashSet 中会出现异常行为。

## ArrayList 和 LinkedList 有什么区别？

ArrayList 基于动态数组，随机访问快、缓存局部性好，尾部追加通常具有摊还 O(1) 性能。LinkedList 基于双向链表，按索引访问慢，节点对象开销大。即使有中间插入需求，也要考虑查找位置的成本，工程中 ArrayList 更常见。

## ConcurrentHashMap 如何支持并发？

现代 Java 的 ConcurrentHashMap 通过 CAS、桶级同步和并发控制减少全表锁竞争。读操作通常不需要获取全局锁，扩容也支持多个线程协助迁移。它不允许 null 键和值，以减少并发语义歧义。

## volatile 能保证什么？

`volatile` 保证变量写入对其他线程可见，并限制相关指令重排序，但不能让复合操作自动具备原子性。例如 `count++` 包含读、改、写三个步骤，仍需要锁或原子类。

## 线程池为什么比频繁创建线程好？

线程池复用线程，减少创建销毁开销，并通过队列和最大线程数控制并发，防止资源耗尽。设计时需要根据任务性质选择核心线程数、队列、拒绝策略和超时机制，不能无界扩张。

## Java 强引用、软引用、弱引用有什么区别？

强引用存在时对象通常不会被回收。软引用在内存压力下可能被回收，曾用于内存敏感缓存，但行为不够可预测。弱引用只要没有强引用就可被回收，常用于不会阻止对象生命周期的映射或监听关系。

## JVM 堆和线程栈分别保存什么？

堆由线程共享，主要存放对象实例和数组，是垃圾回收重点区域。每个线程有自己的 Java 虚拟机栈，栈帧保存局部变量、操作数栈和方法调用信息。栈过深可能触发 StackOverflowError，堆不足可能触发 OutOfMemoryError。
